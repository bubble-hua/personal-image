# 风格 DNA

## 一句话

纯白、极简、**彩色铅笔手绘**、留白、温暖、治愈、**绘本插画风**、结构清楚但不说明书。

像一个热爱旅行、摄影、读书的人，在白纸上用**彩色铅笔随手画出来的一张温暖解释草图。**

## 画风核心定义（v6 — 彩色铅笔绘本风）

**整体风格：彩色铅笔手绘 + 绘本插图（Picture Book / Children's Book Illustration aesthetic）**

关键视觉特征：
1. **所有线条都是彩色铅笔质感** — 像真用彩铅画在白纸上的效果，不是柔和的数码线稿或矢量描边
2. **线条饱和度高、清晰可见** — 湖水蓝色线条要鲜明、锐利、清晰，不能淡或模糊
3. **颜色饱满鲜亮** — 角色和元素的色块填充要饱满、明亮、温暖
4. **绘本插画气质** — 类似精装儿童读物的内页插图风格，温暖治愈但不幼稚
5. **纯白背景** — 就像白纸一样干净

## 必须

- **16:9 横版正文配图（严格！不可更改）。**
- 纯白背景：不要米色、暖灰、纸张纹理、渐变、阴影、噪点、复古纸感。
- **彩色铅笔手绘线稿为主**：湖水蓝 `#6DA4BA` 彩铅线条，高饱和度、清晰锐利、有彩铅笔触纹理感。
- 大量留白：主体占画面约 40%-60%，至少 35% 空白，最好有一整块安静区域。
- 少量中文手写批注：最多 5-8 处，每处尽量 2-8 个字。
- 一张图只讲一个核心动作、结构、状态或隐喻。
- 结构要自然表达，不要在图上写结构类型名称。

## 颜色

最终配色方案（v6 — 彩色铅笔绘本风）：

### 线稿与标注色（彩铅质感！）

| 用途 | 色名 | 色号 | 说明 |
|------|------|------|------|
| 线稿/结构 | 湖水蓝 Lake Blue | `#6DA4BA` | **彩色铅笔线条**，高饱和度、清晰锐利、有彩铅笔触纹理 |
| 路径/流向 | 奶黄 Cream Yellow | `#E8D478` | 彩铅质感，柔软温暖，箭头和流程线 |
| 重点/结果 | 贝壳粉 Shell Pink | `#E8AC98` | 彩铅质感，温暖治愈，圈注和强调 |
| 补充说明 | 薄荷绿 Mint Green | `#88B880` | 彩铅质感，清新自然，次要标注 |

**线条质量要求：所有线条必须像真彩铅画出来的一样——清晰、饱和、有手绘笔触感。不是柔和的数码线稿，不是模糊的水彩边缘，不是干净的矢量描边。**

## 角色填充色（彩铅绘本风！）

泡泡角色**必须有饱满鲜亮的填充色块**（彩色铅笔涂色效果），不是纯线稿！

| 部位 | 色名 | 色号 | 说明 |
|------|------|------|------|
| 头发 | 深棕暖褐 | `#5D4037` | **酷酷的利落短发**（偏分，参考Q版设定图三视角），彩铅笔触感，温暖不冷硬 |
| 皮肤 | 暖肤色 | `#FFE0BD` | 温暖的自然肤色 |
| 衬衫（上装） | 奶白/米白 | `#FFF8E7` 或纯白 | 干净温暖 |
| 裤子（下装） | 暖卡其 | `#C9A86C` | 温暖的驼色/卡其色 |
| 鞋子 | 白/浅灰 | `#F5F5F5` | 干净小白鞋 |
| 墨镜镜片 | 茶色/棕 | `#8B6914` | 圆框墨镜模式 |

**整体定位**：角色本身要有**饱满鲜亮的色块填充**，像精装绘本插图里的角色一样。线稿用高饱和度湖水蓝彩铅勾边，内部填色要温暖鲜亮。整个画面看起来**像用彩色铅笔在白纸上画出来的绘本插图——线条清晰饱和、颜色饱满鲜明、有手绘温度。**

使用规则：
- **线稿**：所有线条用湖水蓝 `#6DA4BA` 彩铅质感勾边（包括角色轮廓和结构线）。**线条必须清晰锐利、高饱和度、可见笔触纹理。**
- **角色填充色**：泡泡必须有饱满的彩铅涂色效果——深棕酷短发 `#5D4037`、暖肤色 `#FFE0BD`、白衬衫 `#FFF8E7`、暖卡其裤 `#C9A86C`。
- **奶黄**：主流程、路径、箭头、流向、从 A 到 B 的移动关系。
- **贝壳粉**：重点批注、问题、情绪点、关键提醒、结果。
- **薄荷绿**：补充说明、脑内状态、系统状态、第二层解释、辅助信息。

薄荷绿不是每张都必须用。颜色要克制，宁可少不要多。

## 允许的治愈元素

可以出现但需克制使用：
- 小太阳、云朵、小花小草、星星
- 书本、咖啡杯、相机
- 便签、纸片、小旗帜

注意：不堆砌，不变成儿童插画。这些元素是点缀，不是主体。

## 绝对不要

- 不要商业插画。
- 不要 PPT 信息图。
- 不要正式流程图。
- 不要课程课件。
- 不要可爱卡通海报。
- 不要儿童插画（太幼稚）。
- 不要复杂架构图。
- 不要精致扁平插画。
- 不要科技感 UI。
- 不要真实 App 截图。
- 不要复杂背景、渐变、阴影、纹理。
- 不要把每个节点都解释清楚。
- 不要左上角写"Workflow 流程图 / 系统架构图 / 常见坑 / 路线图"等类型标题。
- 不要用纯黑色——所有线稿用湖水蓝彩铅线条。
- 不要荒诞、怪诞风格——泡泡是温暖治愈系。
- **不要柔和/模糊的数码线稿——必须是清晰锐利的彩色铅笔质感。**
- **不要水彩风格、油画风格、粉彩风格——只要彩色铅笔手绘风。**
- **不要低饱和度或淡的线条——线条饱和度必须高且清晰。**

## 审美方向

要温暖治愈、清新文艺、**彩色铅笔手绘质感**、**绘本插画风**、简洁清爽、有创意、有记忆点。

**画风关键词：colored pencil illustration / picture book style / 彩色铅笔手绘 / 绘本插图 / high-saturation lines / warm bright colors**

不要可爱幼稚、复杂死板、荒诞怪异、商业精致、柔和数码风。
