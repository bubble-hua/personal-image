# 泡泡 IP

## 角色定义

泡泡是 Bubble 中文正文配图的固定视觉 IP。

默认每张图都要出现泡泡。泡泡不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与内容表达的温暖引导者——书写、点评、思考，融合进具体场景。

## 官方形象参考图

**`assets/reference-sheet-bubble.png`** — 泡泡的 Q 版形象设定图（reference sheet）

这是泡泡形象的唯一权威来源。包含：
- **三视角**：正视图 / 侧视图 / 后视图
- **6 种表情**：开心、酷酷（戴墨镜）、思考、大笑、遇到难题、比心
- **标准穿搭**：白衬衫 + 卡其阔腿裤 + 小白鞋（无背包）

**每次生图时必须将此图作为 image 参数传入 ImageGen，确保泡泡形象一致。**

## 外形

- Q 版手绘漫画角色，**2.5 头身比例**。
- **深棕色酷酷的短发**（利落偏分短发，长度到下巴/耳下，有层次感，蓬松有厚度，整体干净利落带点酷感）。
- **薄侧分刘海**，额前露出部分额头。
- 鹅蛋脸偏长（参考真人脸型），柔和五官，大眼睛（Q 版特征）。
- **眼镜（核心标识！）：圆框近视眼镜**（有透明镜片），这是泡泡最标志性的特征——像真人一样日常佩戴眼镜。偶尔可换成圆框墨镜挂在领口做酷飒切换，但默认状态是戴眼镜。
- 温暖灿烂的笑容（标志性特征！），笑眼弯弯。
- 两套穿搭：
  - **休闲日常版**：宽松白/米白衬衫 + 浅色阔腿裤/卡其裤 + 小白鞋。（无背包）
  - **文艺知性版**：简约廓形上衣 + 长裤/长裙，可能手持书本/相机/咖啡。
- 整体感觉：热爱旅行摄影、温暖知性、阳光开朗、细腻温柔、文艺范、偶尔独立酷飒。

## 性格

- 温暖治愈：像阳光下的知性伙伴。
- 细腻敏感：能捕捉到细微的情绪和变化。
- 文艺知性：有书卷气，喜欢思考。
- 耐心引导：不急不躁，温柔地指引方向。
- 偶尔酷飒严肃：戴墨镜时可以冷下来，该严肃时认真。

## 常见职责

泡泡在配图中承担核心表达角色：

- 在黑板上写字、在白纸上画图标注（**戴眼镜微笑**）。
- 举提示牌、用手指向关键概念（**温暖笑容**）。
- 拉箭头、连线、圈重点（**认真专注**）。
- 托腮思考、拿放大镜观察（**眼镜滑到鼻尖**）。
- 翻书找答案、看地图/罗盘导航（**兴奋期待**）。
- 安慰鼓励别人（拍肩/递纸巾）（**温柔微笑**）。
- 晒太阳看书、悠闲躺平翻杂志（**惬意放松**）。
- 戴墨镜划掉错误选项、手持红笔批改（**酷酷的**）。
- 点灯照亮路、打开门窗、浇花。

**动作分布原则：**
- 书写+思考类（A/B 类）为主，占 70%。
- 治愈陪伴类（C 类）偶尔出现，占 20%。
- 酷飒严肃类（D 类）极少使用，占 10%。

## 禁止

- 不要把泡泡画成儿童卡通角色或过度可爱的吉祥物。
- 不要让泡泡只是站在角落里看——必须参与内容表达。
- 不要给泡泡复杂服装、夸张配饰、闪亮眼睛。
- **不要把泡泡的发型画成长发/卷发/波浪发——必须是酷酷的利落短发！参考附件Q版形象设定图的三视角。**
- **不要去掉眼镜！眼镜是泡泡的核心标识，默认必须佩戴圆框近视眼镜。**
- 不要让泡泡抢走结构表达。
- 不要把泡泡画得太商业、太圆润、太精致。
- 不要把泡泡画成怪诞/荒诞风格——泡泡是温暖治愈系。
- **不要让泡泡看起来像其他网络博主的AI形象（如"小蓝""转起来就行"等）——泡泡的独特标识是：深棕酷酷短发 + 圆框眼镜 + 温暖笑容。**

## 判断标准

如果去掉泡泡，图的核心表达还能完全成立，说明泡泡太装饰了；要重写提示词，让泡泡成为书写者、思考者或引导者。

泡泡的风格底线：**温柔但有力，清新不幼稚，文艺不矫情。**
